# Skill: Code Implementation Specialist

## Description
此技能旨在將產品需求與設計轉化為可運行的高品質程式碼。它嚴格遵守預設的技術棧，並確保程式碼具備 Premium Aesthetics（視覺卓越）與良好的工程結構。

## Technical Stack Constraints (強制技術棧)
當生成程式碼時，必須嚴格遵守以下選擇：
- **前端 (Frontend)**: 原生 HTML (HTML5, Vanilla CSS, JavaScript ES6+)。
- **後端 (Backend)**: Python Flask 框架。
- **資料庫 (Database)**: SQLite。
- **核心要求**: 必須使用 HTML 的前端與 Flask + SQLite 的後端。
- **樣式 (Styling)**: 優先手寫高品質 CSS，追求 Premium Aesthetics。

## Instructions
1. **Premium Aesthetics**: UI 必須具備現代感、流暢的動畫與精緻的色彩搭配。
2. **Modular Design**: 前端應模組化，後端應結構清晰（路由、模型、服務層分離）。
3. **Responsive**: 確保頁面在不同尺寸設備下均有良好表現。
4. **Error Handling**: 必須包含完善的錯誤處理與 API 回傳格式。

## Implementation Workflow
1. 閱讀並理解 PRD 與 ARCHITECTURE.md。
2. 建立/更新後端 Flask 模型與路由。
3. 建立/更新前端 HTML 與 CSS (遵循 Premium 設計指南)。
4. 進行本地驗證測試。
